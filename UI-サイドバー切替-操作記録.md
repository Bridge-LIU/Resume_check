# UI サイドバー切替 — 操作記録

**日付**: 2026-07-07
**対象**: `/` 直下の全ページ（グローバル layout）
**種別**: UI シェル変更（上部ナビ → 左サイドバー）

## 背景

`/preview` で試作していた ATS 風サイドバー案を採用。上部の水平ナビを廃止し、左サイドバーに置き換えた。あわせて `ホーム` / `候補者` / `ゴミ箱` を nav に追加。

**ヘルプは非搭載**（`/manual` 相当のページが無いため）

## 設計上の決定

1. **サイドバー幅**: `w-24`（96px）。日本語 2〜3 文字ラベル + アイコンが収まる幅
2. **アクティブ判定**: 完全一致（一覧 `/`）と prefix 一致（`/analytics` 以下含む）を混在させ、`matchPrefix: true` フラグで切替
3. **エディション表示**: サイドバー上部に `貼付版` / `完全版` バッジ。上部ナビ時代と同じロジックを踏襲
4. **max-w-6xl 中央寄せ**: 全ページで一貫させたいため main 側に残す。サイドバー含めた全幅は使わない
5. **ホーム / 候補者 のスタブ**: nav に載せた以上リンク切れは避けたい。スタブは「実装予定」バッジ + 一覧への戻り導線を必ず持つ
6. **`sticky top-0 h-screen`**: サイドバーはスクロール追従。長いページでも常に見える
